# Modern Glassmorphism Portfolio

A modern personal portfolio website designed using a "blue glass" (glassmorphism) aesthetic. The project is developed using HTML, CSS, and vanilla JavaScript, focusing on clean design, responsiveness, and user experience.

---

## Project Overview

This portfolio website represents the work and technical profile of **Deva Nandan R**, a B.Tech student specializing in Artificial Intelligence and Data Science.

The design incorporates modern web development practices such as glassmorphism, responsive layouts, and modular UI components to deliver a visually appealing and functional interface.

---

## Key Features

- Glassmorphism-based user interface with translucent panels and gradient backgrounds  
- Fixed full-screen background image for a consistent visual experience  
- Responsive design compatible with mobile, tablet, and desktop devices  
- Interactive modal windows for displaying project details  
- Structured technical skills section categorized by domain  
- Smooth navigation and basic UI interactions using JavaScript  

---

## Technologies Used

- **HTML5** – Semantic structure and content organization  
- **CSS3** – Layout design, animations, and styling (including glassmorphism effects and grid system)  
- **JavaScript (Vanilla)** – UI interactions such as navigation behavior and modals

## Setup Instructions

To run this project locally, follow the steps below:

1. Clone the repository:
   ```bash
   git clone https://github.com/DNR5995/Portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```

3. Open the project:
   Open the `index.html` file in any modern web browser.

## Running the Project

### Local Execution
The project can be run directly in a browser. For improved development experience, it is recommended to use the Live Server extension in Visual Studio Code.
